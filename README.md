# Investigar ¿Que es el pip y porque lo actualizamos?

pip es el gestor de paquetes de Python (su nombre significa "Pip Installs Packages") y es la herramienta que te permite instalar, actualizar y eliminar 
librerías desde el repositorio oficial PyPI sin tener que descargarlas manualmente. Lo actualizamos porque cada nueva versión corrige errores, mejora 
la compatibilidad con paquetes recientes, soluciona vulnerabilidades de seguridad y optimiza el rendimiento general, por lo que tenerlo desactualizado
puede causar instalaciones fallidas o comportamientos inesperados.

9. Activamos el entorno virtual:

venv/scripts/activate #En windows

10. Hacemos actualización del pip de Python

python3 -m pip install --upgrade pip #Actualizamos el pip
